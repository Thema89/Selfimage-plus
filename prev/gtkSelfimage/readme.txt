3. Introduction
-------------------------------------------------------------------------------
gtkSelfimage is a imaging programm for linux (and windows) like dd or selfimage.
It enables the user to make a zipped bitcopy of a disk (harddrive or usb).


-------------------------------------------------------------------------------
4. Compilation
-------------------------------------------------------------------------------
gtkSelfimage needs the gtk devel libs installed. If this is the case just type make 
in the gtkSelfimage directoy.
